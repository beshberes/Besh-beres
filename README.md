# 🤖 بش برس - هوش مصنوعی فارسی

## ویژگی‌های اصلی
- پردازش زبان طبیعی فارسی
- سیستم یادگیری تعاملی
- رابط کاربری تحت وب
- پایگاه داده SQLite
- مدیریت دانش پویا

## نحوه اجرا
1. نصب requirements: `pip install -r requirements.txt`
2. اجرای سرور: `python app.py`
3. مرورگر: `http://localhost:5000`
